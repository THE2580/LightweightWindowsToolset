# LightweightWindowsToolset — Agent 上下文提示词

## 项目定位
Windows 系统托盘插件式桌面工具集（Electron 33 + React 19 + TypeScript strict）。首期工具为 PC 端游戏资源捕获：快捷键 → 截图 → OCR → DeepSeek AI → 后端 API → 数据库。

## 必读文件
启动前先完整阅读 `E:\codex_agent_project\LightweightWindowsToolset\PROJECT_SUMMARY.md`，该文件包含完整的项目进展、已实现功能、未修复问题、关键设计决策、文件地图和环境信息。

## 核心约束
- **开发命令**: `cd electron-app && npm run dev`
- **构建命令**: `cd electron-app && npx electron-vite build`（TypeScript strict: true）
- **打包命令**: `cd electron-app && npm run package`（仅明确说"打包"时执行，打包前清理 dist 并 kill 轻量化工具集.exe 进程，CSC_IDENTITY_AUTO_DISCOVERY=false）
- **代理**: 所有网络操作通过 `socks5://127.0.0.1:7897`
- **窗口**: 676x444 固定不可调，侧边栏 155px/44px
- **主题**: 蓝白灰（light #2563EB / dark #3B82F6），禁止渐变球/光晕装饰、禁止负 letter-spacing、禁止 UI 卡片嵌套卡片
- **窗口关闭**: 默认直接退出，可选缩小到托盘；托盘右键"退出"必须终止进程
- **AI 聊天**: 内置功能非工具，不在工具列表和开关中
- **中文字符串**: 必须 UTF-8 存储，严禁 PowerShell 管道传中文给 Python
- **已删除文件不要恢复**（如 tavily.ts），已移除功能不要重新实现（如联网搜索、测试模块）
- **设置页**: 编辑时显示保存按钮，保存成功后隐藏（不允许自动保存）
- **工具禁用 = 快捷键禁用**: 通过 tool:set-enabled IPC 通知主进程
- **新工具默认无快捷键**（默认空字符串）
- **快捷键**: 追加式录入，保存时一次性检测，JSON 数组存储。规则：≥2 键、左修饰右普通、仅一个普通键。录入时禁用全局快捷键，保存/取消后恢复
- **Windows 文件写入**: 禁止使用 PowerShell 的 Set-Content / Out-File / [System.IO.File]::WriteAllText 等 shell 写文件方式，必须使用 apply_patch_add_file / apply_patch_update_file / apply_patch_replace_file / apply_patch_batch 工具（参考 `C:\Users\THE\.codex\skills\apply-patch-preferred\SKILL.md`）
